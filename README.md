3. **Skills**: Showcase of technical skills with icons
4. **Projects**: Featured projects with descriptions and tech stacks
5. **Contact**: Contact form and social media links

## Getting Started

Simply open `index.html` in your web browser. No build process required!

### Customization

1. **Personal Information**: Edit the content in `index.html`
   - Change name, title, and bio
   - Update skills and projects
   - Modify social media links

2. **Styling**: Customize colors and styles in `styles.css`
   - Adjust gradient colors
   - Modify card styles
   - Change button appearances

3. **Functionality**: Enhance features in `script.js`
   - Add form submission handling
   - Integrate with backend APIs
   - Add more interactive features
